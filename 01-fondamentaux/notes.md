# Module 01 - Fondamentaux Claude IA

## Vue d'ensemble

Claude est un assistant IA developpe par Anthropic. Cette section couvre les bases indispensables pour comprendre et utiliser Claude efficacement.

---

## Concepts cles

### 1. Les modeles Claude
- Claude Sonnet - Modele equilibre, rapide, ideal pour la majorite des taches
- Claude Opus - Modele le plus puissant, meilleur pour les taches complexes
- Claude Haiku - Modele leger et ultra-rapide pour les taches simples

### 2. Capacites multimodales
Claude peut traiter et analyser du texte, des images, des fichiers PDF et generer des graphiques via du code.

### 3. Les Artifacts
Les Artifacts sont des sorties structurees que Claude peut creer : pages HTML, code, SVG, Markdown, jeux simples.
Ils s'ouvrent dans un panneau dedie a droite de l'interface.

### 4. Les Projects
Equivalent des GPTs. Permettent des instructions personnalisees persistantes, conservent le contexte, peuvent inclure des fichiers de reference.

### 5. L'API Claude
Accessible via console.anthropic.com. Permet d'integrer Claude dans des apps tierces. Dispose d'un Prompt Generator integre.

---

## Prompting - Niveaux

### Prompt Simple
Une instruction directe. Ex: "Explique-moi le machine learning en 5 points cles."

### Prompt Avance
Structure avec : Role + Mission + Format + Ton + Contraintes.

---

## Bonnes pratiques

- Definir un role pour orienter la reponse
- Preciser le format attendu
- Iterer et reformuler
- Utiliser les Projects pour les workflows recurrents
- Utiliser le Prompt Generator pour optimiser
